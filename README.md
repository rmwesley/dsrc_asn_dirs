# Dependencies
- pycrate

# Setup
Create a Python virtual environment named **.venv**: `python -m venv .venv`.
Or, if you are using VSCode, just press "Ctrl+P" and execute the command "Python: Create Environment...".

Then install pycrate in Linux, run `.venv/bin/python -m pip install pycrate`
Or in Windows, execute `.\.venv\Scripts\python.exe -m pip install pycrate`
